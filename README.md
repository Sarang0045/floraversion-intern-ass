# FloraVision

FloraVision is a premium indoor plants landing page built with React, TanStack Start, Vite, and Tailwind CSS. It presents a dark botanical storefront experience with animated hero content, product cards, customer reviews, plant highlights, and a footer subscription section.

## Features

- Responsive FloraVision landing page
- Animated sections using Framer Motion
- Dark green glassmorphism visual style
- Product showcase for indoor plants
- Custom hero plant card with shaped glass border
- Cloudflare-ready server entry

## Tech Stack

- React 19
- TanStack Start
- TanStack Router
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React icons
- Cloudflare Wrangler

## Getting Started

### Prerequisites

- Node.js 20 or newer
- npm

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The app will start on the local Vite development URL shown in the terminal.

### Build For Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```

### Format

```bash
npm run format
```

## Project Structure

```text
src/
  assets/                 Plant images and background assets
  components/
    floravision/          Landing page sections and cards
    ui/                   Reusable UI primitives
  hooks/                  Shared React hooks
  lib/                    Utility and error handling helpers
  routes/                 TanStack route files
  router.js               Router setup
  server.js               SSR/server entry
  start.js                TanStack Start setup
  styles.css              Global styles and Tailwind setup
```

## Main Files

- `src/routes/index.jsx` composes the home page.
- `src/components/floravision/Hero.jsx` contains the hero section and featured plant card.
- `src/components/floravision/TopSelling.jsx` contains the product grid.
- `src/components/floravision/TrendyPlants.jsx` contains the trending plant section.
- `src/components/floravision/Footer.jsx` contains the footer and newsletter form.
- `src/styles.css` defines the theme, glass effects, and custom card shape styles.

## Deployment Notes

- `vite.config.js` uses `@lovable.dev/vite-tanstack-config` for the TanStack Start, React, Tailwind, path alias, and Cloudflare build setup.
- `wrangler.jsonc` points Cloudflare Wrangler to `src/server.js`.
- `dist/` is generated by the production build and should not be edited manually.
